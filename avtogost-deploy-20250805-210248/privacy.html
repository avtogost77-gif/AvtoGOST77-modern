Политика конфиденциальности

1.1. Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сайта avtogost77.ru (далее — «Сайт») компанией АвтоГОСТ (далее — «Компания»).

1.3. Используя Сайт и предоставляя свои персональные данные, Пользователь выражает согласие с условиями настоящей Политики.

2. Персональные данные

2.1. Персональные данные — любая информация, относящаяся к прямо или косвенно определенному физическому лицу.

2.2. Компания обрабатывает следующие персональные данные:

  • Фамилия, имя, отчество
  • Номер телефона
  • Адрес электронной почты
  • Название организации
  • Адреса отправки и получения грузов
  • Технические данные (IP-адрес, данные браузера, cookies)

3. Цели обработки персональных данных

3.1. Персональные данные обрабатываются в следующих целях:

  • Предоставление логистических услуг и грузоперевозок
  • Связь с клиентами по вопросам заказов
  • Расчет стоимости доставки
  • Контроль исполнения обязательств по перевозке
  • Ведение документооборота по заказам
  • Улучшение качества услуг
  • Маркетинговые исследования

4. Правовые основания обработки

4.1. Правовыми основаниями обработки персональных данных являются:

  • Согласие субъекта персональных данных

5. Способы обработки персональных данных

5.1. Обработка персональных данных осуществляется:

5.2. К обработке персональных данных относятся: сбор, запись, систематизация, накопление, хранение, уточнение, использование, передача, обезличивание, блокирование, удаление, уничтожение.

6.1. Персональные данные могут передаваться:

6.2. Передача персональных данных осуществляется только в объеме, необходимом для достижения целей обработки.

7.1. Персональные данные обрабатываются в течение срока достижения целей обработки.
