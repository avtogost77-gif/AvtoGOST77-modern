## 🚀 **БЫСТРЫЙ СТАРТ**

### 📦 **Установка зависимостей:**
```bash
npm install -g sass
```

### 🔧 **Компиляция SCSS:**
```bash
# Разработка (с source maps)
sass styles/main.scss:compiled/main-dev.css --style=expanded --source-map

# Продакшн (сжатый)
sass styles/main.scss:compiled/main-production.css --style=compressed --no-source-map
```

### 📁 **Структура проекта:**
